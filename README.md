# Gomes Jardinagem

## Objetivo

Desenvolver um site institucional para uma empresa fictícia de jardinagem e paisagismo.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Estrutura

- Home
- Sobre
- Serviços
- Contato

## Funcionalidades

- Navegação entre páginas
- Layout responsivo
- Exibição dinâmica de informações dos serviços
- Ano automático no rodapé
- Menu hambúrguer para dispositivos móveis
- Destaque da página atual no menu
- Botão "Voltar ao topo"
- Validação do formulário de contato
- Exibição de erros de preenchimento
- Aviso visual para campos inválidos
- Mensagem de sucesso após envio
- Limpeza automática do formulário após envio
